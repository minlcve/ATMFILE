# ATM Machine Program

## Description

This is a simple ATM machine program that allows a user to:

- Create a bank account with a user ID and password.
- Login using their credentials (user ID and password).
- Perform basic banking operations once logged in, such as:
  - Deposit Money
  - Withdraw Money
  - Request Balance
- Quit the program.

The program runs in a loop, allowing users to create an account, log in, and perform actions until they decide to quit.

## Features

- **Create Account**: A user can create a new account by entering a user ID and password.
- **Login**: Users can log in with their existing user ID and password.
- **Banking Actions** (after successful login):
  - Deposit money into the account.
  - Withdraw money from the account.
  - Request the current account balance.
- **Quit**: Exit the program at any time.

## Instructions

1. **Running the Program**:
   - Compile and run the program to start.
   - You'll be greeted with a menu with the following options:
     - `l` -> Login
     - `c` -> Create New Account
     - `q` -> Quit the program

2. **Login**:
   - If you choose `l`, you will be prompted to enter a user ID and password. If the credentials are correct, you will be granted access to perform banking actions.
   - If the login fails, you will be prompted to try again.

3. **Create New Account**:
   - If you choose `c`, you will be prompted to enter a user ID and password to create a new account.

4. **Banking Actions**:
   - After logging in, you can choose to:
     - Deposit money (`d`)
     - Withdraw money (`w`)
     - Request balance (`r`)
     - Quit the program (`q`)
